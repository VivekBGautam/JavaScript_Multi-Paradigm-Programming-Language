# Assignment 6

This folder contains all the warmup programs. Each program performs the following tasks: 

program6_1.js – Takes a name as input from the user and displays it on the screen

program6_2.js – Checks whether the given number is greater than 100 or not

program6_3.js – Checks whether the given number is greater than 100 or not (second variation)

program6_4.js – Multiplies three numbers and handles the case where any number is zero

program6_5.js – Calculates percentage using total marks and obtained marks

How to Execute the Programs 

Use the following commands in the terminal to compile and execute the .js files:

node filename.js
